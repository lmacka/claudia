Break-glass

This is the emergency path. Use it only when Jasper has forgotten his passphrase, or in a genuine safety situation where you need to read past sessions.

What happens when you do this

  • You enter the printed envelope key from setup.
  • If you're using this to reset Jasper's passphrase: you'll be prompted to enter a new one. All of his existing sessions get re-wrapped with the new passphrase. Old key is invalidated.
  • If you're using this to read past sessions: an audit event is written. Next time Jasper logs in he sees a banner: "Your dad used the emergency key on YYYY-MM-DD to access past sessions. Talk to him about why if you want to."
  • The banner persists until Jasper clicks "I've seen this." There's no way to suppress it. This is intentional and non-negotiable.
